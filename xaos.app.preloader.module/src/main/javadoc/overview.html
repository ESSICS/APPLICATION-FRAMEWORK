XAOS (ΧΑΟΣ – the primordial state of existence) is a JavaFX application framework.

The framework was initially conceived to create accelerator physics applications based on Open XAL, an open source development environment used for creating accelerator physics applications, scripts and services.

The framework is compose by a series modules: the XAOS App Preloader module provides the preloader (i.e. splash-screen) part of the application framework for building UI-based applications. The module is separated from the xaos.app to provide a faster JAR file to be loaded as first one in the application run-time life-cycle.
