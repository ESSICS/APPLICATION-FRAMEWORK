Allows pan and zoom operations to be performed using keyboards accelerators.

Action Accelerator
Pan Down Shortcut+DOWN
Pan Left Shortcut+LEFT
Pan Right Shortcut+RIGHT
Pan Up Shortcut+UP
Redo Shift+Shortcut+Z
Undo Shortcut+Z
Zoom In Shift+Shortcut+UP
Zoom Out Shift+Shortcut+DOWN
Zoom To One Shortcut+EQUALS

Where SHORTCUT stands for CTRL on Windows and Linux, and COMMAND on macOS.
